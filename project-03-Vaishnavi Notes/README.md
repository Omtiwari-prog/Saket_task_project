# 📝 Notes Manager (Python)

A simple command-line Notes Manager built using Python. This application allows users to create, view, search, update, and delete notes, making it an excellent beginner-friendly file handling project.

## 📌 Features

- ➕ Add New Notes
- 📖 View All Notes
- 🔍 Search Notes
- ✏️ Update Existing Notes
- ❌ Delete Notes
- 💾 Save Notes to a Text File
- 📂 Persistent Storage using File Handling
- ⚡ Simple and Easy-to-Use Menu Interface

## 🛠️ Technologies Used

- Python 3.x
- File Handling
- Functions
- Loops & Conditional Statements

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Notes-Manager.git
```

2. Navigate to the project folder

```bash
cd Notes-Manager
```

3. Run the application

```bash
python notes_manager.py
```
