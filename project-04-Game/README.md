# 🎯 Number Guessing Game (Python)

A fun and interactive command-line game built with Python where the computer randomly selects a number, and the player tries to guess it within a limited number of attempts. The game provides hints if the guess is too high or too low.

## 📌 Features

- 🎲 Generates a random number
- 🔢 User enters guesses
- ⬆️ Hint if the guess is too high
- ⬇️ Hint if the guess is too low
- 🎉 Congratulates the player on a correct guess
- 🔄 Play again option
- 📊 Tracks the number of attempts

## 🛠️ Technologies Used

- Python 3.x
- Random Module
