# 📱 Contact Management System (Python)

A simple command-line Contact Management System built using Python. This application allows users to add, view, search, update, and delete contacts while storing the data in a file for future use.

## 📌 Features

- ➕ Add New Contact
- 📋 View All Contacts
- 🔍 Search Contact by Name
- ❌ Delete Contact
- 📂 Persistent Data Storage
- ⚡ Easy-to-use Menu Interface

## 🛠️ Technologies Used

- Python 3.x
- File Handling
- Functions
- Lists & Dictionaries

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Contact-Management-System.git
```

### 2. Navigate to the Project Folder

```bash
cd Contact-Management-System
```

### 3. Run the Application

```bash
python contact_management.py
```
